# Contributing

Thanks for your interest in contributing to Claude Usage!

## Getting Started

1. Fork the repo and clone it
2. Run `./build.sh` to build
3. Make your changes in `ClaudeUsage.swift`
4. Rebuild and test manually (no automated test suite)

## Guidelines

- Everything lives in a single file (`ClaudeUsage.swift`) â€” please keep it that way
- No external dependencies or Xcode projects
- Test on macOS 12+ if possible
- Keep the app lightweight and focused on its purpose

## Submitting Changes

1. Create a branch for your change
2. Open a pull request with a clear description of what you changed and why
3. Include a screenshot if your change affects the UI

## Reporting Issues

Open an [issue](https://github.com/Kevinjohn/claude-usage-swift/issues) with:
- What you expected to happen
- What actually happened
- Your macOS version
